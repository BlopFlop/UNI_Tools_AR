# UNI_Tools_AR
Плагин, расширяющий функционал для приложения Revit.

# Функционал
- UpdateLegends. Обновление легенд, ищет те легенды(View) на котором расположено 
специальное семейство рамка(подгружается автоматически из Resources),
делает из этого вида картинку и вставляет в выбранный вами параметр у элемента.
- CountCoefficient. Расчет коэффициентов, получает последнее расчетное значение из выбранных спецификаций, 
и по формуле расчитывает необходимое вам значения, итоговое значение записывается в 
глобальный параметр. Кнопка обновление значений автоматом перерасчитывает уже заданные значения.
- CreateFinishWall Отделка стен, позволяет создать на основе геометрии помещения, отделку стен.
- CreateFinishFloor Отделка полов, позволяет создать на основе геометрии помещения, отделка перекрытий.
- CreateFinishCeiling Отделка потолков, позволяет создать на основе геометрии помещения, отделку потолков.
- CountInsolation Расчет инсоляции, на основе геометрии солнца, позволяет подсчитать инсоляцию для окна.

# Инструкция
- Для запуска проекта необходимо установить Revit 2021/22/23
- Итоговые .dll файлы и файл .addin вставить в папку Addins(Для локального пользователя C:\Users\{CurrentUser}\AppData\Roaming\Autodesk\Revit\Addins, для глобальной установки C:\ProgramData\Autodesk\Revit\Addins)
- Либо запустить установочник setup.exe
